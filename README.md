The Image Gallery is a responsive web application developed using HTML, CSS, and JavaScript. It allows users to browse a collection of images through an interactive and visually appealing interface. The project includes Next and Previous navigation buttons, a Lightbox (popup) view for enlarged image display, smooth transition effects, and hover animations. The layout automatically adjusts to different screen sizes, making it compatible with desktops, tablets, and mobile devices. Additionally, optional image filtering or category selection can be added to organize and display images based on user preferences.

Technologies Used
HTML5 – Structure of the gallery.
CSS3 – Styling, responsive design, and animations.
JavaScript – Image navigation, lightbox functionality, and interactive features.
